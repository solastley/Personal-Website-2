Data Structures

The purpose of this class was to discuss the fundamental data structures in computer science as well as their most popular implementations in the Java programming language. Additionally, some algorithm analysis was touched on.

Discrete Structures

In this class, we discussed a wide variety of topics in basic discrete mathematics and probability. In particular, we discussed how these topics pertain to the world of computer science. Some topics include logic, mathematical reasoning (proof strategies), relations, and counting.

Differential Equations

This class discussed primarily linear, first-order, and second-order differential equations. Solution techniques for both homogeneous and inhomogeneous differential equations were discussed. Computer Organization and Assembly Language

This class presented a low-level view of a computer processor. We had many projects that were programmed in MIPS assembly language, and we spent a significant amount of time designing our own computer processor which could implement a simplified version of MIPS.

Calculus 3

A typical calculus 3 course. Some topics that were discussed in detail include functions of several variables and their representations in space, partial derivatives and their applications, and multiple integrals and their applications.
