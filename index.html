Sign Me

This is a project that my team completed at Dragon Hacks 2016 at Drexel University. It won for best hack overall. The project was actually a hardware hack in which we utilized a Microsoft Kinect and an open-source algorithm for sign-language translation to facilitate communication between deaf and non-deaf individuals.

With Sign Me, a non-deaf individual could wear a motion-tracking camera which would record their partner speaking in American sign language. Sign Me would then translate the signs to English and send the translation to the user via a text message. The user could then respond to their partner via a voice-to-text message.

Emergency Law Inventory

I was the primary developer for this project while working at MAYA Design as a software engineering intern. The Emergency Law Inventory (ELI) is a website where medical reserve corps emergency respondents can go if they are being deployed to another state which has different liability and scope of practice laws for their profession. The website prompts the user for their profession, location, and scenario (state of emergency or not) and then displays emergency-related laws from a database that have been translated into layman's terms by members of the Pitt law school.

The website runs on a primarily jQuery driven front-end with a PHP backend that uses a content management system called Kirby, the same CMS I used to make my personal blog. It connects to a PostgreSQL database that I built from scratch with a Python script that hits the Google Drive API to collect data from a form the Pitt law school had been filling out for data input. I later built an extension of the website that acts as a portal for the law school to edit, add, and remove laws in the future.

MediRoute

This is a project that my team completed at Prof Hacks 2016. It won three awards, including Best Use of Amazon Web Services and Best Internal Integration. The theme of this hackathon was "The Physical Web," and the projects were meant to involve sensors that improved some aspect of our lives with technology.

MediRoute is an application for self-driving cars with passengers that have high levels of risk for things such as strokes, seizures, and heart attacks. If the passenger were to experience a medical emergency while in their self-driving car, MediRoute sensors would detect this and re-route the car to the nearest hospital. It would then send the emergency information and medical history for the patient to the hospital so the personnel could be ready for the passenger's arrival.

By utilizing the Muse Headband, we were able to read the passenger's brain waves. If the readings fell below a minimum threshold, then a medical emergency was detected. The application screen would change and ask the passenger if a medical emergency was indeed occurring, giving them the option to signal a false alarm in case of a misreading. If the passenger did not respond within ten seconds, the car would be re-routed to the nearest hospital.

Recursive Maze Solver

This is a project I completed for my Data Structures class. The idea of the project was as follows: create an algorithm which takes a maze input file as a command-line argument and solves that maze using recursion and backtracking.

Essentially, the algorithm attempts every single possible path in the maze and prints out a solution each time it finds one, all the while keeping track of the shortest solution. At the end of the program it prints the shortest solution and the total number of recursive calls made.
